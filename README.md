### alpha 0.1：

### alpha 0.3：
- 实现了 时间戳和url的解析
- 实现了 蓝图+json+dataclass的互相转换
- 把字符串改成了大写的R(心虚)

### alpha 0.5
- 加了 应用/临时转换 脚本。这个脚本是给没代码基础的人看的。

- 强化了 应用/临时转换 脚本。

- 修复了parameter的部分问题，并修改了部分逻辑。
- 修复了输入时没转int32的bug。
